# NativeScript TypeScript Application for weather forecasting
Nativescript application for weather forecasting.

# Dependencies
* Nativescriot 3.1.3+
* Typescript 2.1.2+

# How to build 
`tns build android`

# Features
1.  Current weather
2.  5-days weather forecasting
3.  UV-index

# Roadmap
